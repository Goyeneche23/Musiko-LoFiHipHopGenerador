# Musiko-LoFiHipHopGenerador

Sequence-based neural network built with TensorFlow using an embedding layer followed by stacked LSTM layers. The model is designed to learn patterns from tokenized musical data and generate sequences by predicting the next token in a sequence. It uses dropout for regularization and is trained with categorical cross-entropy for multi-class prediction over a discrete vocabulary.

Note: The repository does not contain the full dataset or trained model due to size limitations. External links are provided for access to these resources.
